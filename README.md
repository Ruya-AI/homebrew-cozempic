# homebrew-cozempic

Homebrew tap for [Cozempic](https://github.com/Ruya-AI/cozempic) — context cleaning CLI for Claude Code.

## Install

```bash
brew install Ruya-AI/cozempic/cozempic
```

Or tap first, then install:

```bash
brew tap Ruya-AI/cozempic
brew install cozempic
```

## Upgrade

```bash
brew upgrade cozempic
```

## Uninstall

```bash
brew uninstall cozempic
brew untap Ruya-AI/cozempic
```
